# Personal Calisthenics V0.1

Date: 2026-06-08

Status: active test

## Goal

Use current capacity without chasing fatigue: preserve heart-health training, hot yoga recovery, mobility, stability, posture, and bone-health support.

This protocol assumes the current working capacity includes about 50 clean regular push-ups and 50 clean bodyweight squats in a focused session without harming running.

The active A/B sessions avoid lying on the back. Supine work is allowed once weekly as its own block so the standing sessions do not require getting down mid-session.

| Day | Protocol |
| --- | --- |

| Wednesday | Night 5K |

| Saturday | Easy cardio, optional equipment layer or supine block |
| Sunday | Recovery walk, optional legs/bone/balance module if recovered |

## A/B Ladder

Use one A option on Monday and one B option on Friday.

- Beginner is the fallback for deloads, poor recovery, or form checks.
- Intermediate is the current default test.
- Advanced is a low-volume probe when recovery, joints, running, and hot yoga are clean.

## Targeted Modules

Use only 1 module on a given day.

### Core Stability And Moving Plank

- Hip Rotations (Push-up Position): 2 x 5-8 each side.
- CKC Parascapular Exercises: 2 x 2-4 holds of 5-10 seconds each side.
- Modified Side Plank: 2 x 20-30 seconds each side.

### Back And Posture

- Downward-Facing Dog: 2 x 30-45 seconds.
- Bird-Dog: 2 x 8 each side.

### Legs, Bone, And Balance

- Forward Lunge: 2 x 6-10 each side.
- Standing Calf Raises - Wall: 3 x 15-20.

## Separate Equipment Layer

Use 1 time per week as the normal dose. A second exposure should be short and easy: pull-up practice or carries only.

This layer covers the current no-equipment gaps: true pulling and external load.

### E1 Pull + Loaded Carry

| Pattern | Movement | Dose |
| --- | --- | --- |
| Vertical pull | Pull-ups or Chin-ups | 3-5 clean submax sets, 1-3 reps in reserve |
| Loaded squat | Goblet Squat | 2-3 x 8-12 |
| Loaded carry | Suitcase Carry | 3-5 short carries or 20-40 seconds each side |
| Vertical press | Single Arm Overhead Press | 2-3 x 5-8 each side |

Notes:

- Keep this separate from A/B unless the week is unusually easy.
- Do not add this if Wednesday 5K, Thursday hot yoga, or Friday B already feels like enough.
- Kettlebell hinge and swing work remains a later research candidate until the exact exercise-library reference and starting dose are settled.

## Once-Weekly Supine System

Do this as one intentional floor block. Get down once, finish the block, then stand up.

### S1 Supine System

| Pattern | Movement | Dose |
| --- | --- | --- |
| Pelvis/core | Supine Pelvic Tilts | 1-2 x 6-10 slow reps |
| Posterior chain | Glute Bridge Exercise | 2 x 8-12 |
| Integrated core | Half Turkish Get-up | 2 x 1-3 each side, light or unloaded |

## Progression

Progress only if the current week has clean recovery.

Progression order:

1. Keep the same volume but improve smoothness and range.
2. Reduce rest while form stays clean.
3. Add reps to accessories.
4. Add a third targeted module.
5. Add harder variations.

Do not add push-up or squat volume until the 50/50 baseline is boring and recovery is clean for at least two weeks.

## Regression

Reduce by 25 to 40 percent if:

- Wednesday 5K feels worse than normal.
- Thursday hot yoga feels unusually draining.
- Leg soreness changes gait, stairs, or running.
- Shoulder, wrist, knee, hip, or back pain appears.
- Morning floor starts feeling like training fatigue instead of preparation.
