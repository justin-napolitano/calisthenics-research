# SP-0001: Running And Cardio Heart-Health Plan

Status: active
Parent: EP-0004

## Goal

Maintain enough aerobic work for heart health while keeping running sustainable.

The current thesis to test is documented in [Cardio Thesis](../research/cardio/2026-06-08-cardio-thesis.md).

## Current Anchors

- Wednesday night 5K.
- Saturday easy cardio.
- Additional walking or low-impact cardio as needed.

## Research Questions

- How much running is enough for heart-health benefit?
- How much weekly cardio should come from running versus walking or low-impact work?
- What log signals indicate running is exceeding recovery?

## Review Data

- Weekly moderate-equivalent minutes.
- Running minutes.
- 5K effort.
- Lower-leg soreness or pain.
- Sleep and recovery notes.
