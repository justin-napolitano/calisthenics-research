# EP-0001: Atomic Movement Framework

Status: deferred pending EP-0003
Date: 2026-06-08

## Goal

Create a file-backed movement atom framework that can later become a database.

This work is deferred. EP-0003 must first ground the equilibrium thesis, vocabulary, and pressure tiers against the literature.

## Deliverables

- Movement atom schema.
- Seed movement atoms.
- Seed skill records.
- Seed transfer links.
- Domain map.
- Critical review of the framework.

## Work Packages

1. Define atom fields and status model.
2. Seed cross-domain atoms.
3. Seed representative skills from calisthenics, running, yoga, grappling, boxing, and Muay Thai.
4. Add transfer links.
5. Review for overclaiming, missing evidence, and poor boundaries.

## Acceptance Criteria

- A new skill can be described with the atom model without inventing a new top-level field.
- Evidence-backed claims are not mixed with personal hypotheses.
- Transfer links are marked as hypotheses unless tested.
- Martial arts content remains non-mystical and non-violent in framing.

## Risks

- The atom model may be too abstract to guide practice.
- Transfer claims may overreach before personal testing.
- Public site copy may sound more certain than the evidence supports.
