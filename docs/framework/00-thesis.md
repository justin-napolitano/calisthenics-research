# Framework Thesis

## Core Claim

Health practice and martial arts can be studied as equilibrium management under different pressures.

The first framework is not a database of movement atoms. It is a kinesiology and conflict-dynamics model:

- preserve my own balance, posture, perception, and mobility
- disrupt the other person's balance, posture, base, timing, perception, or recovery options
- create exit, takedown, or immobilization
- use the least harmful option that works at the current pressure tier
- test every claim against literature, live constraints, and personal logs

## Project Goal

Build a framework that turns movement systems into researchable structures that can be:

- described in plain language
- observed in practice
- linked to evidence
- logged over time
- compared across domains
- reviewed critically
- stored in a future database after the vocabulary stabilizes

## Domain Scope

Current:

- health-first calisthenics
- running and cardio
- hot yoga and mobility

Planned:

- grappling
- boxing
- Muay Thai

## Translation Pattern

For each skill:

1. Name the visible movement.
2. Identify the intent.
3. Identify the balance or equilibrium problem.
4. Identify the pressure tier.
5. Identify the physical constraints.
6. Link the claim to evidence or working assumptions.
7. Define simple drills only after the claim is grounded.
8. Log personal response.
9. Review for transfer across systems.

## Deferred Implementation

Movement atoms, schemas, and database tables are useful later. They are not the first source of truth. The first source of truth is the literature-grounded equilibrium framework.
