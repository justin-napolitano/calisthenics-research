title: Current Routine
description: The active baseline week for cardio, calisthenics, mobility, equipment work, and one weekly supine block.
---

The baseline routine prioritizes heart health, stability, mobility, bone-health support, and repeatability. Mass and maximum strength are secondary.

## Protocol Snapshot

| Target | Current test |
| --- | --- |
| Cardio | 180-240 moderate-equivalent minutes per week |
| Strength | 2 focused calisthenics days per week |
| Morning floor | 10-15 minutes most mornings |
| Review | First full decision after 4 logged weeks |

## Weekly Schedule

| Day | Focus | Intent |
| --- | --- | --- |
| Monday | Calisthenics A | Joint control, squat, push, posterior-chain control, back/posture, trunk, balance. |
| Tuesday | Easy aerobic base | 30-40 minutes walk, walk-jog, bike, or equivalent. |
| Wednesday | Night 5K | Sustainable run, warm-up, cooldown, no extra hard work. |
| Thursday | Hot yoga | Mobility, balance, breath, heat response, recovery check. |
| Friday | Calisthenics B | Single-leg control, posture, trunk, lateral stability, mobility. |
| Saturday | Easy cardio | 30-45 minutes easy run, walk-jog, bike, or long walk. Optional equipment or supine block if recovery is clean. |
| Sunday | Recovery walk | 30-60 minutes walking plus short mobility. |

## Morning Baseline

Alternate Core A and Core B by day. This is preparation, not training fatigue.

| Pattern | Movement | Count | Reference |
| --- | --- | --- | --- |

| Squat | [Bodyweight Squat](https://www.acefitness.org/resources/everyone/exercise-library/135/bodyweight-squat/) | 6-10 clean reps / 3 minutes | ACE |

## A/B Strength Ladder

Use one Monday A option and one Friday B option. Start with intermediate unless recovery says otherwise. Advanced movements are tests, not the normal workload.

| Tier | Role | Use when |
| --- | --- | --- |
| Beginner | Fallback / form check | Running, morning floor, hot yoga, or soreness already feels like enough. |
| Intermediate | Current default test | Recovery is normal and the session should reflect current capacity. |
| Advanced | Test-only option | Joint response, running, and hot yoga are clean. |

## Targeted Modules

Use these only when the full-body sessions, Wednesday 5K, and Thursday hot yoga are recovering cleanly.

| Module | Cadence | Movements |
| --- | --- | --- |
| Core stability and moving plank | 1-2x/week, 6-10 minutes | Hip Rotations, CKC Parascapular Exercises, Modified Side Plank |
| Back and posture | 1-2x/week, 6-10 minutes | Downward-Facing Dog, Bird-Dog |
| Legs, bone, and balance | 1x/week only if running recovery is clean | Forward Lunge, Standing Calf Raises - Wall |

## Supine Block

This is one intentional weekly floor session. Do the whole block after getting down once, then keep the standing sessions uninterrupted.

| Pattern | Movement | Count |
| --- | --- | --- |
| Pelvis/core | [Supine Pelvic Tilts](https://www.acefitness.org/resources/everyone/exercise-library/7/supine-pelvic-tilts/) | 1-2 x 6-10 slow reps |
| Posterior chain | [Glute Bridge Exercise](https://www.acefitness.org/resources/everyone/exercise-library/49/glute-bridge/) | 2 x 8-12 |
| Integrated core | [Half Turkish Get-up](https://www.acefitness.org/resources/everyone/exercise-library/381/half-turkish-get-up/) | 2 x 1-3 each side, light or unloaded |

## Progression Rule

Progress smoother reps, better range, better balance, consistent weekly completion, and clean recovery before harder variations, more running, or higher intensity.
